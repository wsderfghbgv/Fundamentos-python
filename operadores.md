# Ejercicios de Operadores Matematicos

## Ejercicio 1

**Expresion:** `5 + 3 * 2`

1. `3 * 2 = 6`
2. `5 + 6 = 11`

**Resultado:** `11`

## Ejercicio 2

**Expresion:** `8 / 2 + 4 * 3`

1. `8 / 2 = 4.0`
2. `4 * 3 = 12`
3. `4.0 + 12 = 16.0`

**Resultado:** `16.0`

## Ejercicio 3

**Expresion:** `(7 + 3) * 2 - 5`

1. `7 + 3 = 10`
2. `10 * 2 = 20`
3. `20 - 5 = 15`

**Resultado:** `15`

## Ejercicio 4

**Expresion:** `10 - 4 + 2 * 3`

1. `2 * 3 = 6`
2. `10 - 4 = 6`
3. `6 + 6 = 12`

**Resultado:** `12`

## Ejercicio 5

**Expresion:** `(10 / 2) * (3 + 2) - 4`

1. `10 / 2 = 5.0`
2. `3 + 2 = 5`
3. `5.0 * 5 = 25.0`
4. `25.0 - 4 = 21.0`

**Resultado:** `21.0`

## Ejercicio 6

**Expresion:** `2 + 3 * (4 - 1)`

1. `4 - 1 = 3`
2. `3 * 3 = 9`
3. `2 + 9 = 11`

**Resultado:** `11`

## Ejercicio 7

**Expresion:** `5 * 2 ** 3`

1. `2 ** 3 = 8`
2. `5 * 8 = 40`

**Resultado:** `40`

## Ejercicio 8

**Expresion:** `6 + 4 / 2 ** 2`

1. `2 ** 2 = 4`
2. `4 / 4 = 1.0`
3. `6 + 1.0 = 7.0`

**Resultado:** `7.0`

## Ejercicio 9

**Expresion:** `10 % 3 + 2 * 5`

1. `10 % 3 = 1`
2. `2 * 5 = 10`
3. `1 + 10 = 11`

**Resultado:** `11`

## Ejercicio 10

**Expresion:** `(8 + 2) * 3 ** 2`

1. `8 + 2 = 10`
2. `3 ** 2 = 9`
3. `10 * 9 = 90`

**Resultado:** `90`

## Ejercicio 11

**Expresion:** `7 + 2 * (3 + 5) / 4`

1. `3 + 5 = 8`
2. `2 * 8 = 16`
3. `16 / 4 = 4.0`
4. `7 + 4.0 = 11.0`

**Resultado:** `11.0`

## Ejercicio 12

**Expresion:** `2 ** 3 * 4 / 2`

1. `2 ** 3 = 8`
2. `8 * 4 = 32`
3. `32 / 2 = 16.0`

**Resultado:** `16.0`

## Ejercicio 13

**Expresion:** `9 - 6 + 3 ** 2`

1. `3 ** 2 = 9`
2. `9 - 6 = 3`
3. `3 + 9 = 12`

**Resultado:** `12`

## Ejercicio 14

**Expresion:** `(7 - 2) * 5 + 3 ** 2`

1. `7 - 2 = 5`
2. `3 ** 2 = 9`
3. `5 * 5 = 25`
4. `25 + 9 = 34`

**Resultado:** `34`

## Ejercicio 15

**Expresion:** `4 * 2 ** 3 / 8 + 1`

1. `2 ** 3 = 8`
2. `4 * 8 = 32`
3. `32 / 8 = 4.0`
4. `4.0 + 1 = 5.0`

**Resultado:** `5.0`
